##  Program 1: Full Name Greeting

### Description
This program asks the user to enter their **first name** and **last name**, combines them into a full name, and displays a friendly greeting message.

### Functionality
- Takes user input for first name
- Takes user input for last name
- Combines both inputs into a full name
- Prints a welcome message using the full name

### Sample Output
```

Enter your first name: Amrithesh
Enter your last name: M T
Hello, Amrithesh M T! Welcome to the Python Program.

```

## Program 2: Basic Arithmetic Operations

### Description
This program takes two numbers from the user and performs basic arithmetic operations on them.

### Operations Performed
- Addition
- Subtraction
- Multiplication
- Division (with zero-division check)

### Functionality
- Accepts two numeric inputs from the user
- Displays the result of each arithmetic operation
- Handles division safely by checking for division by zero

### Sample Output
```

Enter the first number: 10
Enter the second number: 2

Results:
Addition: = 12.0
Subtraction: = 8.0
Multiplication: = 20.0
Division: = 5.0

```

If the second number is zero:
```

Division: Cannot divide by zero

````
